Muhammad Farhan

Web Developer & UI Designer

Sebagai mahasiswa Teknik Informatika di Universitas Muhammadiyah Riau, saya memiliki passion yang mendalam dalam pengembangan website dan desain UI/UX. Perjalanan saya dimulai dari SMKN 2 Pekanbaru jurusan Rekayasa Perangkat Lunak, yang memberikan fondasi teknis yang kuat.


Pengalaman profesional saya mencakup magang di PT Netviro sebagai Web Developer Intern, di mana saya berkontribusi dalam mendesain dan mengembangkan website yang responsif dan user-friendly untuk berbagai klien. Saat ini, saya aktif sebagai freelance web developer, melayani klien dari berbagai industri dengan menggunakan teknologi modern seperti PHP, Laravel, React.js, dan Next.js.


Prestasi yang telah saya raih termasuk menjadi Mahasiswa Berprestasi Universitas Muhammadiyah Riau 2024 dan meraih berbagai juara dalam kompetisi web development tingkat nasional. Saya juga aktif dalam organisasi kemahasiswaan sebagai anggota Divisi Kominfo dan Pengembangan Sumber Daya Mahasiswa dan Riset Teknologi di Himpunan Mahasiswa Teknik Informatika.
